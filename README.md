# WHO_PICO_8
Jeu sur console virtuelle PICO 8.

## Objectif
Apprendre à coder sans dépendre de l’IA à travers la création d’un petit projet concret.

L’objectif principal est de développer des compétences en programmation en expérimentant, en recherchant et en construisant un jeu de A à Z.

---

## Description du projet
Ce projet consiste à créer un jeu au style rétro inspiré de la **Game Boy**, développé sur la console virtuelle **PICO-8**.

Le gameplay s’inspire principalement :
- de **Amnesia: The Dark Descent** (mécaniques de cache-cache / peur / tension),
- avec un rendu visuel et une vue proche de :
  - **Zelda**
  - **Pokémon**

L’univers du jeu est basé sur un monde déjà existant issu d’un **livre horrifique pour enfants** que j’ai écrit.
